## 函式合約

### 函式簽名（生成的 MicroPython）

```python
def _rc_should_keep_wifi_for_ota():
    try:
        import cyberbrick_ota_config as _ota_cfg
        if not (getattr(_ota_cfg, 'SSID', '') and getattr(_ota_cfg, 'OTA_TOKEN', '')):
            return False
        import network as _ota_net
        return _ota_net.WLAN(_ota_net.WLAN.IF_STA).isconnected()
    except Exception:
        return False
```

### 回傳值語意

| 情境 | 回傳值 | RC 後續行為 |
|------|--------|------------|
| OTA config 存在且 STA Wi-Fi 已連上 AP | `True` | 保留 AP channel；不執行 `disconnect()` / `channel` 設定 |
| OTA config 存在但 STA Wi-Fi 未連線 | `False` | RC 正常執行 `disconnect()` + `reconnects=0` + `channel` 設定 |
| OTA config 不存在 | `False` | 同上 |
| `network` module import 失敗或任何例外 | `False` | 同上（安全降級） |
