### 4. JSON 格式降低維護門檻

**設計**: JSON 驅動的白名單配置  
**受益者**: 非程式設計師 (語言學家、翻譯者)  
**成果**: 可直接編輯規則,無需程式碼變更

### 5. 純函數設計提升可測試性
