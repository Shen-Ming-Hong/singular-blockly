# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a [GitHub Security Advisory](https://github.com/Shen-Ming-Hong/singular-blockly/security/advisories/new).
